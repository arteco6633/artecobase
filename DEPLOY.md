# Деплой на Vercel

Проект готов к деплою на Vercel.

## Быстрый деплой через CLI

1. Убедитесь, что вы авторизованы в Vercel:
```bash
vercel login
```

2. Запустите деплой:
```bash
vercel
```

3. Для продакшн деплоя:
```bash
vercel --prod
```

## Деплой через веб-интерфейс

1. Зайдите на [vercel.com](https://vercel.com)
2. Войдите через GitHub/GitLab/Bitbucket
3. Нажмите "Add New Project"
4. Импортируйте ваш репозиторий
5. Vercel автоматически определит настройки для Vite проекта
6. Нажмите "Deploy"

## Настройки проекта

Конфигурация уже настроена в `vercel.json`:
- Build команда: `npm run build`
- Output директория: `dist`
- Framework: Vite

## Примечание

Приложение использует localStorage для хранения данных, поэтому данные будут храниться локально в браузере каждого пользователя. Для production версии с общим хранилищем потребуется backend API.

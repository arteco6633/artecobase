# Проверка деплоя функции на Vercel

## Что проверить

### 1. Файл в репозитории ✅
Файл `api/sync-availability.ts` есть в репозитории (проверено).

### 2. Проверить деплой в Vercel Dashboard

1. Откройте https://vercel.com/dashboard
2. Найдите проект "artecobase"
3. Перейдите в **Deployments**
4. Откройте последний деплой
5. Проверьте:
   - Статус: должен быть "Ready" (зеленый)
   - Логи: не должно быть ошибок
   - Функции: должна быть `api/sync-availability`

### 3. Проверить Functions в Dashboard

1. В проекте перейдите в **Functions**
2. Должна быть функция `api/sync-availability`
3. Если нет - значит Vercel не распознал файл

### 4. Если функция не появилась

Возможные причины:
- Vercel не распознал TypeScript файл как API route
- Нужна другая структура для Vite проектов
- Ошибка компиляции (проверьте логи)

## Решение

Если функция не работает, можно:
1. Переименовать файл в `api/sync-availability.js` (убрать .ts)
2. Или создать через Vercel Dashboard
3. Или использовать другой подход

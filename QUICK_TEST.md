# Быстрое тестирование синхронизации наличия

## Упрощенная функция для тестирования

Так как вам нужно только синхронизировать **наличие**, функция была упрощена.

## Вариант 1: Тест через Supabase Dashboard (без CLI)

1. Откройте: https://supabase.com/dashboard/project/oojntidzikkhaivfdnee/edge-functions
2. Создайте новую функцию `sync-availability`
3. Используйте упрощенный код (см. ниже)
4. Задеплойте
5. Вызовите функцию

## Вариант 2: Использовать существующую функцию

Функция `parse-materials` уже создана, но нужно её задеплоить.

## Упрощенная логика

Функция будет:
1. Читать материалы из вашей БД (таблица `materials`)
2. Для каждого материала искать его на сайте по артикулу
3. Обновлять только поле `availability`
4. Обновлять `last_synced_at`

## Вопрос

У вас уже есть материалы в таблице `materials`? 
- Если да - функция обновит их наличие
- Если нет - нужно сначала добавить материалы

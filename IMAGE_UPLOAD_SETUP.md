# Настройка загрузки фото для таблиц

## Что было добавлено

✅ **Поддержка фото в таблицах:**
- Добавлено поле `imageUrl` в `TableRow`
- Создан SQL скрипт для настройки Supabase Storage
- Добавлена функция загрузки фото
- Обновлен TableEditor с кнопкой загрузки фото
- Обновлен TableViewer для отображения фото

## Настройка

### 1. Выполните SQL скрипт для Storage

1. Откройте Supabase Dashboard: https://supabase.com/dashboard/project/oojntidzikkhaivfdnee
2. Перейдите в **SQL Editor**
3. Откройте файл `database_storage.sql`
4. Скопируйте и выполните скрипт

Это создаст:
- Bucket `table-images` для хранения фото
- Политики доступа (публичный доступ для просмотра, авторизованные пользователи для загрузки)

### 2. Проверьте работу

1. Откройте приложение
2. Создайте или отредактируйте таблицу
3. Нажмите на иконку фото рядом со строкой
4. Выберите изображение
5. Фото будет загружено и отображается в таблице

## Ограничения

- Максимальный размер файла: 5 MB
- Поддерживаемые форматы: JPEG, PNG, WebP, GIF
- Фото сохраняются в Supabase Storage
- Одно фото на строку таблицы
